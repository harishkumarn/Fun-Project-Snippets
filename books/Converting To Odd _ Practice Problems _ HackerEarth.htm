Converting To Odd

Adams discovered the new planet in which he found that it also contains 26 characters represented by lower alphabets.

Where the string formed using these characters have a special property called the odd property in which character has a property of eliminating each other or in other word try to eliminate each other if they are of the same character and next to each other. 

You are given a string s where each s[i] represents a character, where 0i|s|1.

You need to output the size of the final string followed by string after which no elimination is possible.

SAMPLE INPUT
15
aaabbbyybyaaycd
SAMPLE OUTPUT
3
acd
Time Limit: 0.5 sec(s) for each input file.
Memory Limit: 256 MB
Source Limit: 1024 KB
